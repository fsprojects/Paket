Paket

What is Paket?

Paket is a dependency manager for .NET projects. Paket enabled precise and predictable control over your dependencies. With Paket, you can reference:

  • NuGet packages
  • Files from Git repositories
  • Files from any HTTP resource.

Get Started by learning how to use Paket and take a look at the FAQ to learn more about it.

If you are using legacy NuGet (packages.config-based) for package management in your solution, you can learn about the upgrade process in the convert from NuGet tutorial

For information about Paket with .NET SDK, .NET Core and the dotnet CLI see the"Paket and the .NET SDK / .NET Core CLI tools" guide.

Contributing and copyright
